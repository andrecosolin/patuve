# Política de Privacidade - Patuvê

**Nome do app:** Patuvê  
**Data de vigência:** 2026-03-31

## 1. Sobre o app

O Patuvê é um aplicativo que busca vagas de emprego usando IA.

## 2. Dados coletados

O app pode coletar e armazenar, localmente no dispositivo:

- Termos de busca
- Vagas salvas
- Candidaturas registradas

## 3. Armazenamento e envio de dados

- Todos os dados listados acima são armazenados localmente no dispositivo.
- Nenhum dado pessoal é enviado para servidores próprios do Patuvê.

## 4. Uso de links externos

Ao acessar vagas e conteúdos externos, os links são abertos no navegador do dispositivo.

## 5. Uso da API Anthropic

Para realizar buscas com IA, o Patuvê utiliza a API da Anthropic. Dados de busca podem ser processados pela Anthropic conforme a política de privacidade deles.
