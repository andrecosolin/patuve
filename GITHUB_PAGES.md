# Publicar política no GitHub Pages

## 1) Subir o projeto no GitHub

1. Crie um repositório no GitHub (exemplo: `patuve`).
2. Faça commit dos arquivos e envie para a branch `main`.

## 2) Ativar GitHub Pages

1. Abra o repositório no GitHub.
2. Vá em **Settings** > **Pages**.
3. Em **Build and deployment**, escolha:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/docs`
4. Salve as configurações.

## 3) URL gerada

Após alguns minutos, sua política ficará disponível em:

`https://seuusuario.github.io/patuve/privacy-policy.html`

## 4) Usar na Play Store

No Google Play Console, informe essa URL no campo de Política de Privacidade do app.
